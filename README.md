# Trip-Time-Estimator
A machine learning project implemented in R, which visualizes new york taxi trips data and predicts taxi trip duration .

Datasets used in the above projects can be found at the given links-

https://www.kaggle.com/c/nyc-taxi-trip-duration


https://www.kaggle.com/oscarleo/new-york-city-taxi-with-osrm

https://www.kaggle.com/pjoshi15/new-york-satellite-image

https://www.kaggle.com/mathijs/weather-data-in-new-york-city-2016
